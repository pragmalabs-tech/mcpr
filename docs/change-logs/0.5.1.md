# 0.5.1

Release date: 2026-05-02

## Diagnostics

- Per-request `Timer` instrumentation. Each proxy stage (router, request, response, CSP rewrite, schema, session) records its own duration, plus a total request-flow timer. Set `MCPR_DEBUG=1` to dump per-stage timings to stderr at the end of every request.
- Bench harness collapsed to a single tools/call overhead scenario. Report at `benches/reports/20260501-tools-call-overhead.md`.

## Removals

- `mcpr-tunnel` crate moved to its own repo (`mcp-tunnel`). The `mcpr relay` subcommand and the relay config block are gone from this repo.
- Relay sections removed from README, `docs/CLI.md`, and `docs/proxy/PROXY_CONFIGURATION.md`.
