# 0.5.3

Release date: 2026-05-03

## Cloud sink

- `cloud_endpoint` is now optional. If only `cloud_token` is set, the sink defaults to `https://api.mcpr.app/api/ingest-events`.

## Event model

- Sinks no longer capture the raw HTTP request and response bodies. The MCP JSON-RPC payload is still recorded in full; the surrounding HTTP body is dropped to keep the SQLite store small and reduce cloud upload size.
